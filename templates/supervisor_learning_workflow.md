# Supervisor Learning Workflow

Use this workflow to grow supervisor memory over time.

## Inputs

- supervisor papers
- Zotero metadata, abstracts, notes, or BibTeX
- pasted representative writing samples
- before/after revision pairs
- optional local Codex Skill context

## Steps

1. Collect supervisor-authored literature.
2. Parse or paste Zotero-style metadata.
3. Update the supervisor profile in `profiles/`.
4. Collect original/revised draft pairs.
5. Update revision memory in `revision_memory/`.
6. Retrieve relevant memory before future manuscript revision.
7. Revise using public framework memory plus private local Skill context.
8. Add only confirmed and reusable patterns back into memory.

## Generalization Rule

Do not overgeneralize from a single paper, metadata record, or revision pair.
Treat early observations as tentative until repeated across multiple sources.
